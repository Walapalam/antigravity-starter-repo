## üöÄ Getting Started

To get the most out of this repository, ensure you clone it recursively to include all skills:

```bash
git clone --recursive https://github.com/Walapalam/antigravity-starter-repo.git
```

If you've already cloned without submodules, run:

```bash
git submodule update --init --recursive
```
